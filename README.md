# José Simão Neto | Psicólogo e Psicanalista Clínico

Site institucional de José Simão Neto, psicólogo e psicanalista clínico e
consultor organizacional, em Sorocaba (SP). CRP 06/92454-SP.

## Estrutura

Site estático, sem dependências e sem etapa de build.

```
index.html    página única, com todas as seções
styles.css    estilos, incluindo paleta e marca d'água
script.js     menu do celular, revelação ao rolar, ano do rodapé
imagens/      fotos, logotipo e marcas d'água
```

## Identidade visual

Paleta e logotipo do Centro Clínico Campolim: azul-marinho (`#0b1f3a`),
dourado (`#c9a227`) e prata. Tipografia Cormorant Garamond e Inter.

## Como editar

Abra `index.html` em qualquer navegador para ver o resultado. Para trocar uma
foto, substitua o arquivo dentro de `imagens/` mantendo o mesmo nome.
